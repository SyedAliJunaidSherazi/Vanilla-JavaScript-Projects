Features

For civilians who want to travel within the city or between cities. The MASW Travelling Application is a Web-Based and smartphone-enabled application that will help users buy, reserve, cancel tickets/seats based upon the comparative prediction in choosing means of transport that the system provides. Unlike the current online travelling application people who use our application will not have to go to railway station, bus depot and airports for tickets/seats reservation. Also, users can book and reserve rides for travelling within the city premises. Which will ultimately save them time, money and from traffic delays and will increase the travelling efficiency.

Payment Gateway

  • Multiple payment options
  • Provides multiple authentication/login layers before taking payment
  • Provides facility of SSL certificate (Standard Security Technology)
  • Multiple payment options like Easy-paisa, Jazz cash, Payoneer and bank accounts.

Chat Bot

  • Provides Multilingual capabilities.
  • Provides integration with 3rd-party application.
  • Uses WhatsApp as a channel for communication.
  • Provide responsive answers to users.
  • Help user resolve their queries

Book Tickets

  • Can select tickets from various categories available in all modes of transport.
  • Can check the arrival and departure time of the selected mean of transport.
  • Help user predict which mode is suitable for travel.
  • Can contact appropriate booking firm for confirmation.
  • Can select tickets from various categories available in all modes of transport.

Book-Reserve Ride

  • Select type of transport user wants to book like car, bus or bike.
  • Book a ride after selecting current location and drop-off location.
  • Schedule ride in advance either for future travels or in meantime.
  • Built-in mapping and GPS-tracking capabilities.
  • Asks User Permission to fetch his location.
